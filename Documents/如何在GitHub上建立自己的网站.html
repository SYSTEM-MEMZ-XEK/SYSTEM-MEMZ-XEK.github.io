如何在GitHub上建立自己的网站（AI制作，未完工，后续将增加图文教程）

GitHub是全球最大的开源代码托管平台，可以让你和他人协作开发项目。下面是如何在GitHub上建立自己的网站的步骤：

  1. 注册GitHub账号
  2. 创建仓库
  3. 安装visual studio code和fitten code
  4. 新建HTML、CSS、JavaScript文件
  5. 编辑网页内容
  6. 上传网页内容至GitHub仓库
  7. 注意事项
  8. 视频教程

一、注册GitHub账号

首先，你需要注册一个GitHub账号。你可以访问GitHub官网，点击右上角的Sign up按钮，填写相关信息即可。

二、创建仓库

创建仓库的第一步是登录GitHub账号。登录成功后，点击右上角的Your repositories按钮，然后点击New repository按钮，填写仓库名称、描述、是否公开等信息，点击Create repository按钮即可。

三、安装visual studio code和fitten code

安装visual studio code和fitten code的第一步是访问Visual Studio Code官网和Fitten Code官网，下载安装包。安装成功后，打开Visual Studio Code，点击Extensions按钮，搜索Live Server，安装Live Server插件，点击左下角的Live Server按钮，就可以在浏览器中实时预览网站内容。

四、新建HTML、CSS、JavaScript文件

新建HTML、CSS、JavaScript文件的第一步是打开仓库目录，点击New file按钮，输入文件名称，点击Create file按钮，编辑文件内容，保存文件。

五、编辑网页内容

编辑网页内容的第一步是打开HTML文件，编辑文件内容，保存文件。

编辑CSS文件的第一步是打开CSS文件，编辑文件内容，保存文件。

编辑JavaScript文件的第一步是打开JavaScript文件，编辑文件内容，保存文件。

六、上传网页内容至GitHub仓库

上传网页内容至GitHub仓库的第一步是点击仓库目录，点击Upload files按钮，选择需要上传的文件，点击Commit changes按钮，填写提交信息，点击Commit changes按钮，完成上传。

七、注意事项

创建仓库时，请注意仓库名称，必须是“你的用户名.github.io”的格式，否则无法通过https://你的用户名.github.io访问网站。

不要一次性上传超过100个文件，否则可能会导致上传失败，可以分批次上传。
